# Назначение
Тула для генерации патчей с настройками (внешний ввод), результатом которой должены быть сгенерирован архив, состоящий из 3-х записей:
* Сборка патча
* Сборка SQL скрипта
* SQL скрипт

# Технологии
Основными технологиями являются:
* Spring Boot framework
* Vaadin framework
